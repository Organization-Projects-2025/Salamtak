# Changelog

All notable changes to the Salamtak project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-15

### Added
- Initial release of Salamtak application
- User authentication system (login/signup)
- Municipal problem reporting system
  - Photo upload functionality
  - Location picker with Google Maps and Leaflet integration
  - Voice-to-text description input
  - Report status tracking
  - Report history with filtering
- Safety equipment e-commerce store
  - Product catalog browsing
  - Shopping cart management
  - Order placement and tracking
  - Order history and invoices
- Admin dashboard
  - Report management interface
  - Order management system
  - Product management (CRUD operations)
- Multi-language support (English and Arabic)
- RTL (Right-to-Left) support for Arabic
- Local image storage system
- Firebase integration
  - Authentication
  - Cloud Firestore database
  - Cloud Storage for images

### Removed
- Image classification/recognition features
- AI-based problem type detection
- TensorFlow Lite integration
- Automatic problem categorization from images

### Changed
- Simplified image upload workflow
- Enhanced user interface for report submission
- Improved location picker with dual map support
- Optimized image handling for web compatibility

### Fixed
- Image persistence issues in report system
- Location address formatting
- Voice input language detection
- Report image display in admin panel

### Security
- Implemented secure image storage
- Added proper authentication checks
- Protected admin routes

## [Unreleased]
